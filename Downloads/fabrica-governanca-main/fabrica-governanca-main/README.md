# Fábrica de Software – UCSal

Este repositório centraliza a **governança, processos, padrões técnicos e diretrizes acadêmicas** da Fábrica de Software da Universidade Católica do Salvador (UCSal).

## Objetivo

Padronizar a forma como os projetos são:
- planejados
- desenvolvidos
- versionados
- revisados
- entregues
- mantidos entre semestres

## Estrutura

- `docs/` – processos e padrões
- `templates/` – templates de issues e pull requests
- `POLITICA-ACADEMICA.md` – regras institucionais e acadêmicas
